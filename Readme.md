- Typescript
- Sass
- Prismic.io


<br>

## Instalação e uso

```bash
# Abra um terminal e copie este repositório com o comando
https://github.com/Flaviojcf/space-travelling
# ou use a opção de download.

# Entre na pasta web com 
cd space-travelling

# Instale as dependências
yarn install

# Rode a aplicação
yarn dev
```
